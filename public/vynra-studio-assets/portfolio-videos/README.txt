PORTFOLIO VIDEOS — auto-discovered & auto-categorized
=====================================================

Drop your reels into this folder. On the next build the site picks them up
automatically — no code changes needed.

CATEGORIES
----------
Videos are sorted into the 4 portfolio categories automatically:
  • Business Branding
  • Creator Branding
  • Tech & Educational Content
  • Thought Leadership

Two ways to control which category a video lands in:

1) (Recommended) Put it in a sub-folder named after the category, e.g.:
     portfolio-videos/business-branding/aether-brand-film.mp4
     portfolio-videos/creator-branding/maya-reel.mp4
     portfolio-videos/tech-educational/how-ai-edits.mp4
     portfolio-videos/thought-leadership/the-long-game.mp4

2) Or include a keyword in the file name, e.g. "brand", "creator",
   "tutorial", "podcast". Anything unmatched is distributed evenly.

TITLES
------
The on-card title is generated from the file name:
  "aether-brand-film.mp4"  ->  "Aether Brand Film"

POSTERS (optional)
------------------
Add an image with the SAME base name to set a thumbnail/poster:
  aether-brand-film.mp4
  aether-brand-film.jpg

FORMATS
-------
Video: .mp4 (H.264, recommended), .webm, .mov, .m4v
Vertical 9:16 reels look best, but any aspect ratio works.

Until you add files, the site shows elegant placeholder cards so the layout
is fully visible.
